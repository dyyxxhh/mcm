### What install_game_assets() creates under game root
1. libraries/<artifact.path> — one mock .jar per library with downloads.artifact
2. assets/indexes/<assetIndex.id>.json — mock asset index with sample objects
3. assets/objects/ — empty directory (ready for real asset downloads)
4. versions/<resolved-id>/natives/ — directory for native extraction
5. libraries/<classifier.path> — native classifier jars for libraries with natives map
