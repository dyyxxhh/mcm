## 4. Expected Outcome Checklist

- [x] `/api/auth/oidc/start` creates nonce/state, constructs provider auth URL, stores pending state
  - mock: returns auth_url + login_id + state + mock_user
  - real: builds {issuer}/authorize?... URL with state + nonce, returns auth_url + login_id
- [x] `/api/auth/oidc/callback` validates state, exchanges code for tokens, validates issuer/audience/expiry/nonce, creates session
  - mock: validates state via LoginStore.is_pending(), creates session, completes login
  - real: validates state, POSTs to {issuer}/api/oauth2/token, decodes JWT, validates iss/aud/exp, extracts sub + preferred_username, creates session
- [x] `/api/auth/oidc/poll/{login_id}` supports CLI polling (pending/complete/expired/denied)
  - Pending → returns {"status":"pending"}
  - Complete → returns {"status":"complete","token":"...","owner":"...","expires_at_unix":...} (one-shot)
  - Expired → returns {"status":"expired"} after 10 min TTL (one-shot)
  - Denied → returns {"status":"denied","reason":"..."} (one-shot)
  - Unknown login_id → 404
- [x] `/api/auth/oidc/session` returns current session owner
- [x] `/api/auth/oidc/logout` clears session + cookie
- [x] Mock OIDC only available when MCM_AUTH_MODE=mock (route dispatch by AuthMode)
- [x] Real OIDC mode requires all four OIDC env vars (validated at startup by config)
- [x] Secrets/tokens not logged (SecretString redacted, no eprintln of tokens)
- [x] cargo test passes (26 auth tests + 9 oidc unit tests)
- [x] cargo clippy --all-targets --all-features -- -D warnings passes
