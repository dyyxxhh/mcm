================================================================================
Task 13: Implement server storage outside /x
================================================================================

## Summary
Added SQLite + filesystem blob storage for share packages under a configurable
data directory (default /var/lib/mcm-share, refuses /x). Enforces case-insensitive
slug uniqueness, 2-day post-delete slug reservation, owner-match on update/delete,
overwrite-on-update with no backup. Injectable Clock trait for test time-travel.

## Files created/modified

NEW:
  src/server/storage/mod.rs     - Storage facade, Clock trait, public API, init
  src/server/storage/meta.rs    - SQLite schema, row types, low-level queries
  src/server/storage/blob.rs    - Atomic blob writes (tmp + rename)
  src/server/storage/helpers.rs - Slug normalization, payload validation, /x refusal, time fmt
  tests/server_storage.rs       - 13 integration tests (7 storage-level + 6 HTTP-level)

MODIFIED:
  Cargo.toml                  - added rusqlite (bundled), time parsing feature
  src/server/mod.rs           - Storage wired into ServerState; run_server opens storage
  src/server/config.rs        - removed #[allow(dead_code)] on data_dir field
  src/server/share.rs         - real list/download/publish/update/delete handlers + Owner extractor
  src/lib.rs                  - #[doc(hidden)] pub re-exports for Storage/Clock/outcomes
  tests/server.rs             - updated 3 stub-501 assertions to real-200/404/4xx
