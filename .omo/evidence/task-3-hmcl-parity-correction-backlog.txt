## Vanilla install layout

- versions/<mc_version>/<mc_version>.json
- versions/<mc_version>/<mc_version>.jar
Example: versions/1.20.1/1.20.1.json + versions/1.20.1/1.20.1.jar

## Loader install layout (HMCL-compatible flat)

- versions/<resolved-id>/<resolved-id>.json
- versions/<resolved-id>/<resolved-id>.jar
- versions/<resolved-id>/<loader>-<loader_version>.jar
Example: versions/1.21.1-neoforge-21.1.172/1.21.1-neoforge-21.1.172.json
         versions/1.21.1-neoforge-21.1.172/1.21.1-neoforge-21.1.172.jar
         versions/1.21.1-neoforge-21.1.172/neoforge-21.1.172.jar
