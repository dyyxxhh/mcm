- Client secret is never sent to CLI

## Security Properties

1. CLI only calls server HTTP endpoints — never handles OIDC client secret
2. Only MCM session token stored locally (not provider access/id/refresh tokens)
3. Token never printed in stdout/stderr
4. Session files stored with hashed server URL as filename (SHA-256 truncated)
5. Session file uses atomic write (write to tmp, rename)
