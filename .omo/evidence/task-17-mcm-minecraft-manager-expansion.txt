## Risk / notes

- `/install` is mounted BEFORE the API fallback and SPA fallback, so it won't be caught by the wildcard handler.
- The route is public (no auth required), matching the spec.
- Script env overrides (`MCM_INSTALL_PREFIX`, `MCM_RELEASE_BASE_URL`, `MCM_INSTALL_OS`, `MCM_INSTALL_ARCH`, `MCM_INSTALL_DRY_RUN`) are documented in script comments.
- No `curl | sh` pipe inside the script itself — downloads go to temp, verified with sha256sum, then extracted with tar and copied.
