## QA scenarios
Happy:
- `mcm pkg install ./sample.mcm --yes` → "installed package qa-pkg 1.0.0", files written
- `mcm pkg dl ./sample.mcm --yes` → "downloaded package qa-pkg 1.0.0"
- `mcm pkg make` → valid JSON with schema_version 1
- `mcm pkg list` → "rootmod 1.0.0"
- `mcm install --yes` (auto-select) → "installed package aaa-pkg 1.0.0"
- `mcm do script.mcm --yes` → script runs, marker file created

Failure:
- `mcm pkg install ./sample.mcm` (non-TTY, no --yes) → "confirmation required; pass --yes to proceed", exit 1
- duplicate asset path → "duplicate asset path in package: shaderpacks/dup.zip", exit 1, no partial install
- `mcm install mc1.21.1` → "top-level install does not accept Minecraft smart targets"
- `mcm install sodium` → "top-level install accepts only a `.mcm` file path or URL; raw mod names..."

## Design decisions
1. Split pkg_cmd.rs (dispatch, 138 LOC) from pkg_install.rs (apply logic, 216 LOC) to stay
   under the 250 pure-LOC ceiling. Both files are impl App blocks.
2. pkg_install bridges ModEntry → provider Artifact via mod_entry_to_artifact so the existing
   MockProvider::download path is reused (requires download_url.is_some()).
3. Asset install writes placeholder bytes for embedded/referenced sources (real embedded byte
   extraction is a future task — task 11 mrpack import). Path safety via validate_asset_path.
4. check_duplicate_assets runs BEFORE any file write → atomic abort on conflict.
5. Script execution via `sh -c` with current_dir set to game root. Non-zero exit bails.
6. pkg make excludes local/private data by default (local: None) — secrets never serialized.
7. pkg share confirms via PackageInstall policy, validates target parses, then prints stub message.
8. top_install rejects mc... smart targets and raw mod names before delegating to pkg_install.
9. do_file uses ScriptExecution OperationKind (distinct from PackageInstall) for confirmation.
10. tests/mcm_package.rs: 2 stub-assertion tests updated to reflect implementation; the remaining
    28 tests in that file are untouched (regression net intact).
