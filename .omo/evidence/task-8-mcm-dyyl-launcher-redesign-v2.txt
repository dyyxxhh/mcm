## CLI Commands Implemented

1. `mcm pkg share <file.mcm> --server <url> [--yes]`
   - Validates package, authenticates via OIDC, publishes, prints slug + curl-bash install command

2. `mcm pkg list --server <url>`
   - Lists public packages from server

3. `mcm pkg list --mine --server <url>`
   - Lists packages owned by current authenticated session

4. `mcm pkg update <slug> <file.mcm> --server <url> [--yes]`
   - Updates owned package on server, prints install command

5. `mcm pkg delete <slug> --server <url> [--yes]`
   - Deletes owned package from server

6. `mcm pkg download <slug-or-url> --server <url> [--output <path>]`
   - Downloads package JSON from server

7. `mcm pkg install <slug-or-url> --server <url> --yes`
   - Downloads package JSON and delegates to mcm install
