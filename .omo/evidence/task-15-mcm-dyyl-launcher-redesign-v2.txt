## V2 Lock Schema

```json
{
  "schema_version": 2,
  "kind": "mcm-lock",
  "identity": {"name": "...", "version": "...", "description": "..."},
  "author": {"owner_id": "...", "source": "..."},
  "permissions": {"install": true, "do": false, "full": false},
  "game": {"game": "...", "version": "...", "loader": "..."},
  "steps": [
    {
      "op": "mod.install",
      "permission": "install",
      "args": {...},
      "source_line": "..."
    }
  ],
  "artifacts": [...],
  "created_at": "2024-01-01T00:00:00Z",
  "generator": "mcm"
}
```

## Step Operations

| op | Permission | Description |
|---|---|---|
| game.choose | install | Select game/version context |
| game.install | install | Install game version |
| mod.install | install | Install mod artifact |
| pkg.install | install | Install nested package |
| file.copy | install | Copy artifact to game root |
| file.write | install | Write file to game root |
| net.download | install | Download and verify artifact |
| config.set | install | Set configuration value |
| shell.run | do | Execute shell command |
| mcm.do | do | Nested mcm command |
| root.system | full | System-level command |
