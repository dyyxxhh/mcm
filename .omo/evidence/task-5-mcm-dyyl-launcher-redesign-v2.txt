## Validations Implemented
- Slug validation: [a-z0-9-], 1-64 chars, no consecutive hyphens, not reserved (via validate_package_name)
- Owner check: only owner can update/delete (403 Forbidden)
- Daily push limit: 1 push/day per user (429 Too Many Requests)
- Max 5 packages per user (409 Conflict)
- Delete reservation: 2-day slug reservation for deleting owner
- Install-only validation: rejects actions, launch, local fields (400 Bad Request)
- Body limit: 10MB (413 Payload Too Large)
- Content-Type: application/json required (415 Unsupported Media Type)
