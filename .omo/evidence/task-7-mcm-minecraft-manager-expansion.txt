## Deliverables

### NEW src/confirmation.rs (170 pure non-test LOC)
- ConfirmationPolicy enum: Harmless / Bypassable / NonBypassable
- OperationKind enum: 14 variants (Install, Download, Delete, VersionRemoval,
  PackageInstall, RuntimeInstall, SourceAction, ScriptExecution,
  RootSystemChange, WorldOverwrite, WorldDelete, Autoremove, LaunchOnInstall,
  GameRemove)
- classify(op) -> ConfirmationPolicy
- is_mc_critical(op) -> bool (Autoremove, WorldOverwrite, WorldDelete)
- emit_mc_critical_warning(op) -> prints to stderr
- require_confirmation(op, yes) -> Result<()> (centralized gate)
- confirm_typed(prompt) -> Result<bool> (requires "yes")
- root_escalation_helper(action, interactive) -> Result<()> (sudo/pkexec)
- prompt_yes_no(prompt) -> Result<bool> (pub(crate), shared y/N reader)
- AUTOREMOVE_WARNING constant with required phrases

- confirm_install() now delegates to confirmation::prompt_yes_no via
  classify(OperationKind::Install). Preserves exact "Proceed with install?
  [y/N]" prompt text for backward compat.

### MODIFIED src/lifecycle.rs
- autoremove() emits AUTOREMOVE_WARNING to stderr after --yes gate passes,
  before destructive removal. Preserves exact stdout/stderr for
  characterization tests.

### MODIFIED src/lib.rs

### UNCHANGED src/game_cmd.rs
